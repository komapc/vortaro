  • 📗 Esperanto Wiktionary: Translations extracted from Esperanto Wiktionary dumps.

🌍 What is a "via" translation?

A "via" translation uses an intermediate language (like French or English) to create translations between Ido and Esperanto when direct translations are not available.

Example: If we know Ido→French and French→Esperanto, we can infer Ido→Esperanto via French.
